----------------------------------
|       Arc4u Cyphertool.        |
----------------------------------

Add the certificate file path to the pfx file.

Optionnaly you can add the password to the pfx file.
-p | --password => The password to use for the file pfx certificate

If the password is not set, a prompt will be displayed to ask for the password.


You can decrypt a text or a file by using the following commands:
    text => The text to decrypt
    file => The file to decrypt

Example:

arc4u.cyphertool decrypt from pfx /path/to/certificate.pfx -p password text "cypher text"
arc4u.cyphertool decrypt from pfx /path/to/certificate.pfx -p password file "the path to the file to decrypt."

arc4u.cyphertool decrypt from pfx /path/to/certificate.pfx text "cypher text"
arc4u.cyphertool decrypt from pfx /path/to/certificate.pfx file "the path to the file to decrypt."
